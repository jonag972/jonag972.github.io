Objectifs du Projet Project Objectives

Créer un site portfolio professionnel pour présenter mes compétences, projets et parcours. Architecture Technique Technical Architecture

Le site utilise une architecture statique moderne avec HTML5, CSS3 et JavaScript vanilla. La navigation morphing utilise des transitions CSS pures pour des animations fluides. Le support multilingue (français/anglais) et les thèmes (clair/sombre) sont gérés via JavaScript avec persistance localStorage. The site uses a modern static architecture with HTML5, CSS3 and vanilla JavaScript. The morphing navigation uses pure CSS transitions for smooth animations. Multilingual support (French/English) and themes (light/dark) are managed via JavaScript with localStorage persistence.
