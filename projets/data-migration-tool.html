Contexte du Projet Project Context

Dans le cadre de mon stage au CEA, j'ai développé un outil complet de migration de bases de données Oracle 11g vers PostgreSQL 17.4. Ce projet s'inscrit dans une démarche de modernisation de l'infrastructure informatique et de réduction des coûts de licence. As part of my internship at CEA, I developed a comprehensive tool for migrating Oracle 11g databases to PostgreSQL 17.4. This project is part of an IT infrastructure modernization initiative and license cost reduction strategy.

Fonctionnalités Principales Main Features

L'outil intègre plusieurs fonctionnalités clés : migration automatisée des schémas et données, validation d'intégrité avec comparaison des données source et cible, conversion des objets de base (tables, vues, fonctions, triggers), et génération de rapports détaillés sur le processus de migration. The tool integrates several key features: automated migration of schemas and data, integrity validation with source and target data comparison, database object conversion (tables, views, functions, triggers), and detailed reporting on the migration process.

Architecture Technique Technical Architecture

L'architecture de l'outil repose sur une approche modulaire avec des composants spécialisés. Le module de connexion gère les connexions simultanées aux bases Oracle et PostgreSQL. Le module de conversion utilise Ora2PG pour la conversion initiale des schémas, puis applique des transformations personnalisées pour les cas spécifiques. The tool's architecture is based on a modular approach with specialized components. The connection module manages simultaneous connections to Oracle and PostgreSQL databases. The conversion module uses Ora2PG for initial schema conversion, then applies custom transformations for specific cases.

Technologies Utilisées Technologies Used

Le développement s'appuie sur Python comme langage principal, Ora2PG pour la conversion initiale, psycopg2 et cx_Oracle pour les connexions aux bases de données, et pandas pour l'analyse et la validation des données. L'interface utilisateur utilise Tkinter pour une expérience desktop native. Development is based on Python as the main language, Ora2PG for initial conversion, psycopg2 and cx_Oracle for database connections, and pandas for data analysis and validation. The user interface uses Tkinter for a native desktop experience.

Processus de Migration Migration Process

Le processus de migration suit plusieurs étapes : analyse préliminaire de la base source, génération du schéma PostgreSQL compatible, migration des données avec gestion des types incompatibles, validation croisée des données migrées, et génération de rapports détaillés avec statistiques et recommandations. The migration process follows several steps: preliminary analysis of the source database, generation of compatible PostgreSQL schema, data migration with incompatible type handling, cross-validation of migrated data, and detailed report generation with statistics and recommendations.

Validation et Contrôle Qualité Validation and Quality Control

Un système de validation robuste compare automatiquement les données source et cible. Il vérifie le nombre d'enregistrements, les sommes de contrôle, les valeurs nulles, et les contraintes d'intégrité. Tout écart est documenté avec des recommandations de correction. A robust validation system automatically compares source and target data. It checks record counts, checksums, null values, and integrity constraints. Any discrepancies are documented with correction recommendations.

Résultats et Impact Results and Impact

L'outil a permis de migrer avec succès plusieurs bases de données critiques, réduisant les coûts de licence Oracle de 60% et améliorant les performances de 25% grâce à l'optimisation PostgreSQL. Le temps de migration a été réduit de 80% par rapport aux méthodes manuelles précédentes. The tool successfully migrated several critical databases, reducing Oracle license costs by 60% and improving performance by 25% through PostgreSQL optimization. Migration time was reduced by 80% compared to previous manual methods.
